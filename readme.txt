Для использования нужно иметь на компьютере Python версии 3.6 или выше.
Чтоб избежать повнорной установки библиотек было установлено виртуальное окружение. Чтобы его активировать нужно ввести в корневой папке проекта следующие команды:


для Windows
    venv\Scripts\activate.bat
для Mac и Linux
    source venv/bin/activate



Далее запустить файл bot.py командой python bot.py (Для линукс python3 bot.py). Не закрывать окно во время использования.
